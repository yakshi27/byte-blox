ByteBlox – Console Minecraft-Inspired Block World
Overview

ByteBlox is a console-based, two-player strategy game developed in Java. Inspired by grid-based mechanics similar to Minecraft, the game features a competitive environment where two players take on opposing roles: a Thief and a Police officer.

Gameplay –

The game is played on a 2D grid world:
The Thief must locate and collect three hidden diamonds within a limited number of moves.
The Police has unlimited moves and aims to catch the Thief before all diamonds are collected.
The game follows a turn-based approach, alternating between both players.

Features

Grid-based world implemented using 2D arrays
Role-based gameplay (Thief vs Police)
Diamond collection and inventory tracking
Jump mechanics for advanced movement
Special ability: Diamond shuffle (randomizes diamond positions)
Move-limited gameplay for the Thief
Strategic and decision-based gameplay

Tech Stack

Language: Java
Concepts Used: Object-Oriented Programming (OOP)
Data Structures: Arrays, 2D Arrays, Queue, HashMap, ArrayList
